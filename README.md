# sandy snaps

Your story, our Lens. A browser-based photobooth app built with React + Vite + Tailwind CSS.

## Features

- Live camera access via WebRTC
- Countdown sequence: Get ready → 3 → 2 → 1 → SNAP
- 3 layout options: Classic Strip (4 photos), Duo Grid (4 photos), Trio (3 photos)
- 7 real-time filters: None, Warm, Cool, B&W, Vintage, Fade, Vivid
- 7 color themes for your strip
- Download as PNG or JPG
- Fully responsive

## Tech Stack

- React 18 + Vite
- Tailwind CSS v3
- WebRTC / MediaDevices API
- HTML5 Canvas (strip generation)
- Google Fonts: Rubik Glitch, Edu TAS Beginner, Poppins

## Getting Started

```bash
npm install
npm run dev
```
